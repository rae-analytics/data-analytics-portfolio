# Sales Performance Analysis

This project analyzes a sample sales dataset to identify revenue trends, top-performing products, and category performance.

## Tools Used
- Google Sheets (Excel-style analysis)

## Dataset
- `sales_data.xlsx` contains sample sales transactions:
  - Date of sale
  - Product name
  - Category
  - Quantity sold
  - Price
  - Total_Sales (calculated as Quantity Ã— Price)

## Analysis Performed
- Created a `Total_Sales` column to calculate revenue per transaction
- Summed total sales by product
- Summed total sales by category

## Key Insights
- **Laptops** generated the highest revenue despite lower sales volume.
- The **Electronics** category outperforms Accessories in total revenue.
- Visual charts make it easy to see revenue trends at a glance.
