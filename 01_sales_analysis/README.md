# Sales Performance Analysis

This project analyzes sales transaction data to identify revenue trends,
top-performing products, and category performance.

## Tools Used
- Google Sheets (Excel-style analysis)

## Analysis Performed
- Created a Total_Sales column (Quantity Ã— Price)
- Compared revenue by product and category
- Reviewed monthly revenue performance

## Key Insights
- Laptops generated the highest revenue despite lower sales volume.
- Electronics category outperforms Accessories.
- February had higher total revenue than January.
